This code consists of implementation of Binary Search Tree with the methods to build the BST with the given inputs.



1. BST.java has  add(), remove() ,contains(),get(), iterator(),toArray() methods implemented and called through the main() method.
2.InOrderIterator.java is the class that performs inorder iteration using a stack such that only O(h) space is used, where h is the height of the BST.
3.TreeMapProblems.java has solutions for questions to find pairs with given sum in an array and 
to find numbers that appear exactly once in an array.
